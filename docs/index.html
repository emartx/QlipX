QlipX, Your Snippets Instantly

QlipX is a lightweight macOS menu bar app for saving links, commands, replies, notes, file paths, and other snippets you reuse every day. Open the floating panel instantly, search what you need, and copy it in one click.

  • Global shortcut: Cmd + Shift + Space
  • Local-first storage with no cloud dependency
  • Built natively for macOS with SwiftUI + AppKit

Built for fast reuse, not clipboard history clutter

QlipX focuses on intentional snippet storage: the pieces of text you know you will need again, organized into categories and available from a floating panel above any app.

Quick copy

Save frequently used text and copy it to the clipboard directly from the panel when you need it.

See preview

Floating panel access

QlipX runs in the menu bar and opens as an always-on-top, available from the icon or the global shortcut.

View architecture

Organized categories

Create named categories, collapse groups, search across labels and content, and reorder items when you want a cleaner list.

Core workflow

Private by default

Your data stays on your Mac in a readable JSON file. There is no analytics layer, no sync requirement, and no network dependency.

Get QlipX

Native macOS utility with a simple architecture

QlipX uses a single centralized store as the source of truth. Data flows from the local JSON file into the store and then into SwiftUI views, keeping the app fast, predictable, and easy to maintain.

SwiftUI + AppKit

SwiftUI for views, AppKit for menu bar and floating panel behavior.

Single Store MVVM

One shared QlipXStore handles state, persistence, and business rules.

Debounced persistence

Changes are written locally after short debounce windows to avoid noisy disk I/O.

Minimal dependency footprint

Only one external package is used for the system-wide keyboard shortcut flow.

Built for macOS 13+ with Swift 5.9+, distributed as a direct download, and designed as a menu bar app that stays out of the Dock and app switcher.
